# クイズアプリ
https://quizapp-taupe.vercel.app/

## 概要
問題の作成、保存、編集、削除を行う事ができる4択クイズアプリです。  
4択を2択に絞るボタンを作成しました。
ボタンやアイコンのデザインはMaterial-UIを使用しています。

## なぜ作ったのか
自分で問題を作成できるシンプルなクイズアプリを作ってみたかったため。  
また、ReactとTypeScriptで作成するにあたり、基本的な力が身につくと考えたため。

## 使用技術
- React
- TypeScript
- Material-UI
- Firebase (Cloud Firestore)

## 機能一覧
- 問題作成機能
- 問題編集機能
- 問題削除機能
- 4択を2択に絞る機能
